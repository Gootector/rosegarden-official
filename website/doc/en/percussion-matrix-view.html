Chapter 9. The Percussion Matrix editor

The Percussion Matrix editor is identical to the Matrix editor, except that it shows pitch names on the vertical scale instead of a piano keyboard, and it does not show the duration for each note. This is intended for use with percussion programs on MIDI instruments, in which each note pitch plays a different percussion sound.

The pitch names are only shown if the segment being edited is playing through a percussion instrument that has an associated key mapping to define the names; otherwise the standard matrix view piano keyboard will appear instead. See Percussion Key Mappings for more information about key mappings.
