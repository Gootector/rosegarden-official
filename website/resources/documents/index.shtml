Documents

Rosegarden comes with some documentation in the form of a help system and tooltip help, and you may find the online tutorial (still a work in progress) helpful. Here you can find a handful of other miscellaneous documents.

MIDI Device Program Maps

For those of you looking to create voice mappings for your own MIDI hardware, here is a mini-HOWTO for creating voice definitions for Rosegarden-4. You should also check out the ever-growing device library (a snapshot of which is included with each Rosegarden-4 release) to make sure your device hasn't already been mapped.

Translation

For anyone interested in helping out with translations, here is a translation mini-HOWTO.

Developer Documentation for Rosegarden-4

Automatically generated class hierarchies and extracted code notes can be found on our daily Doxygen dump. Useful for reference and as an introduction to some parts of the code.

There are also some notes to be found in Rosegarden CVS itself.

General MIDI and Audio information

For general information on how to set up your home studio you can consult Bownie's DIY Guide to Recording. It's a bit Windows-centric still but it has some useful information for beginners to the world of home recording.
