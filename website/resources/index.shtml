Help

Here you can find documentation and useful links for users of Rosegarden.

  • Frequently asked questions about Rosegarden. If you have any problems installing or using Rosegarden, please read these first.
  • The Rosegarden Companion book, and a free online tutorial, by D. Michael McIntyre.
  • The Rosegarden Handbook, the reference manual and help text for Rosegarden available from within the program.
  • Have you found a bug? Please read these guidelines to learn the proper procedure for filing a bug report. Thanks!
  • Do you want to request a new feature? Please read these guidelines for suggestions about how to improve the chances that someone will implement your idea.
  • Links to other sites including composers who use Rosegarden, other software, and Linux audio community sites.
  • Various other documents for users and tweakers.
