Using Rosegarden

A Guide to Rosegarden-4 0.9.0





This is currently still halfway between 0.8.5 and 0.9.0 due to time constraints. It documents a number of the most important new features, but much of it has not yet been updated to reflect the current state of things, and what was unfinished before is still unfinished. I didn't want to release it like this, but I want you folks out there in userland to benefit from the new studio info in particular, so I'm sending this up half baked.

Also note, this table of contents is currently somewhat broken. It didn't generate properly, and I don't have time to go dig up an HTML reference in order to try to fix the blasted thing manually.

I hope I can get this finished very soon now. Thanks for your patience, and enjoy Rosegarden!
