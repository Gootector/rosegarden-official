7 Notation

So far, we have largely ignored Rosegarden's notation editing capabilities. While Rosegarden's initial release lacks some key features that are necessary for full-fledge score editing, it is nevertheless a powerful and friendly notation editor.

Rosegarden stores events in a format that's very notation friendly. Notation is such an integral part of the package that I like to think of the notation as the "real" data, though it is, of course, only one of the ways the data can be presented to you.
