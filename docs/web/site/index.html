What is Rosegarden-4?

Rosegarden-4 is the new development series of the Rosegarden MIDI and audio sequencer and musical notation editor. It is free software for Unix and X, running within the KDE desktop environment.

If you want to try it out then please go to the download page for more information.

Current Release

The latest release of Rosegarden-4 is the rather confusingly-numbered 0.1.5, released on 02 May 2002. Significantly this release adds KDE3 and ALSA support.

The current release also includes complete or partial support for MIDI file I/O, playback and recording, a track editor, score display and editing, and a piano-roll/matrix style editor.

Changes since 0.1.4 include

  • KDE3 support (builds from same source tree as KDE2 version)
  • ALSA support for MIDI playback and recording (0.9.0beta12 and above)
  • Notation View "Finale style" note entry - notes sound before entry
  • Triplet and tuplet editing in the Notation view
  • Event editor dialog
  • Tempo dialog accessed through double click on current tempo, new tempo changes can be inserted or overall tempo changed
  • Internal reworking and tidying of command framework for improved undo/redo capability
  • Keyboard accelerators for Transport controls shared across all clients (you can control playback from client windows)
  • Song state information (including Studio parameters) saved to Composition file
  • Playback Latency can be modified through Configuration dialog
  • Instrument program names appear on Track buttons when selected
  • Bigger, brighter song position pointers
  • MatrixView keyboard now plays notes
  • MatrixView Events are coloured by velocity
  • Quick copy of Segments (ctrl and click/drag on Segment)
  • Segments are labelled according to Track
  • Soloing Tracks now works
  • Notation and Matrix clients are labelled against the current file
  • Notional JACK audio support (not working)

Have a look at some pictures of it in use.

You can download the source package from SourceForge, or obtain more recent source code directly from the CVS repository. In order to build it, you will need a Unix-like operating system, the gcc-2.9x C++ compiler, KDE 2.2 with Qt2 or KDE3 wth Qt3. If you haven't got KDE, see the KDE web site.

The SourceForge site also contains some code documentation and a mailing-list.

Contribute

We're always looking for people willing to participate in the project. If you're interested, read the development page.

Rosegarden-2.1

Rosegarden-4 is not yet a stable release product; it is intended for developers, contributors and other interested parties. If you want a proper version of Rosegarden that works, but that looks old and has a rather limited design, try Rosegarden-2.1.
