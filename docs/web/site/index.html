What is Rosegarden-4?

Rosegarden-4 is an attractive, user-friendly audio and MIDI sequencer, score editor, and general-purpose music composition and editing application for Unix and Linux. It is currently somewhere approaching beta quality, following two years of sustained active development.

In latest news: Rosegarden-4 is featured in the February 2003 issue of Sound on Sound, the UK's top professional recording magazine. And have you noticed our Russian translation?

Rosegarden-4 is Free Software under the GNU GPL. It is a KDE application, which means to build it you need the KDE 3.x libraries. However, once installed it can be run within the KDE environment or any other desktop or window manager environment.

Note: if you have a Linux distribution that includes a package simply called rosegarden, you almost certainly have Rosegarden-2.1. (We always recommend that Rosegarden-4 is packaged as rosegarden4.) Consider upgrading!

Current Release

The latest release of Rosegarden is Version 4, Release 0.8.5, released on 18th December 2002. Have a look at some pictures of it in use, download it, or read the Frequently Asked Questions page.

This is still an early, development release, but it is entirely usable by interested users and developers. It features complete or partial support for:

  • Score, piano-roll and track overview editors
  • MIDI and audio playback and recording using ALSA and JACK
  • Audio plugin support using LADSPA
  • MIDI file I/O and Csound export
  • High-quality score printing by exporting Lilypond files

Changes since the previous Rosegarden-4-0.8 release include:

  • Improved MIDI file I/O (better support for banks and merging in import, better support for delay, transpose etc in export)
  • Banks/Programs editor dialog (including importing/exporting of Studios)
  • MIDI Panic Button (for clearing down stuck notes)
  • Added some keyboard controls (though not yet enough) to matrix
  • Added real-time segment delays
  • progress display completely overhauled
  • ALSA clients can be added dynamically (you can start iiwusynth after you've started Rosegarden and it'll appear in the Instruments list)
  • MIDI events filter dialog for MIDI THRU and MIDI record
  • MIDI/ALSA recording bug fixed (stopping after so many recorded events)
  • many bugs, tweaks and performance improvements

Rosegarden-2.1

Rosegarden-2.1 is a quite separate and much older program, containing a reasonable score editor and a reasonably useless sequencer with limited integration between the two. It's very useful for certain tasks, particularly on older hardware, but Rosegarden-4 is where all the fancy new stuff is at.
