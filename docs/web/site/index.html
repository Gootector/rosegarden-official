What is Rosegarden?

"Probably the closest native equivalent to Cubase® for Linux" -- Sound on Sound

Rosegarden-4 is an attractive, user-friendly audio and MIDI sequencer, score editor, and general-purpose music composition and editing application. It is a powerful professional application that runs on Linux and is suitable for composers, musicians, music students, and small studio or home recording environments.

Current Release

The latest release of Rosegarden is Version 4, Release 0.9.5 released on 27th November 2003. Have a look at some pictures of it in use, download it, or read the Frequently Asked Questions page.

This is still a pre-beta release, but it is entirely usable by interested users and developers. It features complete or partial support for:

  • Score, piano-roll, event list and track overview editors
  • MIDI and audio playback and recording using ALSA and JACK
  • Audio plugin support using LADSPA
  • Score interpretation of performance MIDI data
  • MIDI file I/O and Csound export
  • High-quality score printing by exporting Lilypond files
  • Shareable device (.rgd) files to ease MIDI portability
  • Translations into Russian, Spanish, German, French and Welsh

New features since 0.9.1 include:

  • Shared-memory-based sequencer comms level for much more robust MIDI sequencing
  • Control rulers for editing velocity and control events
  • Controller management for MIDI devices
  • Event list editor with some actual editing capabilities
  • Event filter on current selection
  • Classification of audio plugins using liblrdf
  • Percussion and variations support in bank management
  • Import Studio from File
  • Markers
  • File merging for all file types
  • Notation rendering using real scalable fonts (one supplied)
  • Multi-page notation rendering with panner
  • Much better score printing
  • Print Preview
  • Segment colouring
  • Configurable metronome
  • Better Lilypond export
  • MusicXML export that more-or-less works
  • Can now drag notes around in notation view
  • Several new .rgd device files
  • Better handling of MIDI devices that come and go
  • Much improved audio scheduling infrastructure
  • Ability to drag-and-drop audio files onto segment canvas
  • Many optimisations throughout

Rosegarden-4 is Free Software under the GNU GPL. It is a KDE application, which means to build it you need the KDE 3.x libraries. However, once installed it can be run within the KDE environment or any other desktop or window manager environment.

Note: if you have a Linux distribution that includes a package simply called rosegarden, you might have Rosegarden-2.1. (We always recommend that Rosegarden-4 is packaged as rosegarden4.) Consider upgrading!

Rosegarden-2.1

Rosegarden-2.1 is a quite separate and much older program, containing a reasonable score editor and a reasonably useless sequencer with limited integration between the two. It's very useful for certain tasks, particularly on older hardware, but Rosegarden-4 is where all the fancy new stuff is at.
