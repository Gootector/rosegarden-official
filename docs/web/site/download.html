Rosegarden-4 binary downloads

The Rosegarden team currently provides code in source form only. The binaries and RPMs listed below are all maintained by other people, so it may take a short while before they are brought up-to-date with each new release of Rosegarden. Please take a moment to check whether you are really getting the latest release (currently Rosegarden-4-0.8.5) before you download a binary package.

Rosegarden-4 for SuSE

  • Rosegarden-4 for SuSE 8.0 and 8.1 are available from this page on apps.kde.com. Registration required.

Two of the three core Rosegarden developers use SuSE and so you should be able to compile Rosegarden out of the box on a SuSE 8.0 or 8.1 distribution. (If you have 8.1, consider installing SuSE's JACK and LADSPA packages before you start.)

Rosegarden-4 for RedHat

  • Rosegarden-4 for RedHat 7.3 and 8.0 from Planet CCRMA.

    Planet CCRMA is a well-maintained and easy to use distribution of audio and sound applications for RedHat, which can install everything you need semi-automatically. If you want to set up an audio workstation on RedHat, your best bet is to install the Planet CCRMA system: read the documentation on the Planet site, and go from there.

If you want to run Rosegarden-4 on RedHat but you don't want to use Planet CCRMA, you're probably on your own. It's a difficult distribution to run much modern audio software on, because it doesn't include the ALSA 0.9.x sound drivers. You may prefer to contact RedHat and ask when they intend to include Rosegarden-4 in their distribution.

  • Rosegarden-4 is available as part of Debian unstable, thanks to Enrique Robledo Arnuncio.

Rosegarden-4 for your distribution

It looks as though Rosegarden-4 should be officially included as part of future releases of SuSE, Mandrake and Debian. If we're not packaged as part of your favourite distribution then please get in touch with your distribution manufacturer and ask why not! If you've made an RPM or package for any distribution then please let us know so that we can link it here.

Rosegarden-4 source code

If you can't find a package to suit your system them you can still build Rosegarden-4 from source code. The latest release is shown below:

  • Rosegarden-4 release 0.8.5 sources, 18 December 2002, for download from the SourceForge page.
  • Rosegarden-4 current sources from the SourceForge CVS repository; this is work-in-progress and might not build or run.

The SourceForge site also contains some code documentation and a mailing-list.

See the requirements page for details on how to build the source code.

Design reports

Rosegarden started life as a university project. These are the original reports Chris and Andy produced for that project, way back in the first half of 1994. They probably still constitute the best description of the internals of the Rosegarden 2.1 application, although many details are now out-of-date.

  • Notation editor design document (212K, gzipped PostScript).
  • Sequencer design document (691K, Acrobat PDF).
  • Interlock tool-communication framework design document (96K, Acrobat PDF).
