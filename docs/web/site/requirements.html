Requirements

Rosegarden is a MIDI and audio sequencer and notation editor. System requirements for editing, playing and recording MIDI are usually much lower than those for use with audio. Use of audio plugins especially will impact on your system performance. Bear this information in mind when specifying your machine.

Minimum system requirements for running Rosegarden for mainly MIDI use with simple editing and playback/recording is a 400MHz system with 64MB of memory. Recommended system is an 800MHz system with 256MB of memory - with this setup you should have enough power to sequence MIDI and audio with a number of audio plugins.

If you want to hear sound from your system then you will also need to configure a soundcard supported by the ALSA project. If you're using an external MIDI device (sound module or keyboard) then you'll also need the appropriate leads and cables. For help setting up your home studio for easy audio and MIDI recording then you can consult Bownie's DIY Guide to Recording.

Please also consult the FAQ for more information.
